Optional attributes

duration

Sets the animation duration in miliseconds. Default is 800. It's very important to set this to lower than the shortest expected value change interval.

easing

Sets the easing method. This can be any of the Robert Penner's easing functions. Default it "linearEase".

precision

Sets how precisely should floating numbers be displayed. Default is 0 (display as rounded integers). This parameter is used only when no filter is selected. Setting this to -1 disables rouding completly.

filter

Sets which angular filter the directive should use for display. Default is none. If a filter requires parameters, they should be passed like that: number:3.

Setting default values

If you have a lot of instances of the directive and they all are supposed to behave the same, setting their options via attributes may become troublesome. This is where the trainsiterateDefaultsProvider comes in. It enables you to set default values for all directive instances across your application/page. They can still be overriden with attributes when needed.

The provider is available for injection in the app.config() block. The values are set with the .setDefault(option, value) method. The directive then uses the service internally to read the values.

angular.module('myApp', ['ngTransiterate']) .config(function(transiterateDefaultsProvider) { transiterateDefaultsProvider.setDefault('duration', 3000); });

You can set any of the options described above and chain calls to the .setDefault(option, value) method, like so:

angular.module('myApp', ['ngTransiterate']) .config(function(transiterateDefaultsProvider) { transiterateDefaultsProvider .setDefault('duration', 6000) .setDefault('easing', 'easeOutQuad') .setDefault('filter', 'number:3'); });
